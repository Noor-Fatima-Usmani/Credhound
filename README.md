##  Project Description

**CredHound** is a PyQt5-based forensic analysis tool designed for authorized security auditing and local system inspection. It scans files and directories to detect potentially exposed sensitive information such as emails, API keys, authentication tokens, password hashes, and other secret patterns.

It also includes a browser artifact analysis module and provides structured forensic report generation for documentation and review purposes. The tool is intended strictly for **educational use and authorized cybersecurity research / penetration testing environments only**.
