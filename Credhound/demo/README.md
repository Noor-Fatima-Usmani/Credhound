

##  Project Demo Overview

This project provides a user-friendly forensic analysis interface where users can select the type of file they want to recover. Based on the selected file type, the system automatically scans and fetches relevant files from the target source for further analysis.

In addition to file recovery, the tool includes a **Browser Artifacts** module. This feature extracts important browser-related data such as stored credentials and other forensic artifacts that can assist in investigation and analysis.

The project also supports **automated report generation**, which compiles all extracted data into a structured and readable report, making documentation and evidence analysis easier and more efficient.
